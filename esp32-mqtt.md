# ESP32 MQTT 一键集成 Skill

## 功能概述

为 ESP32 Arduino 项目**一键集成 MQTT 联网功能**。

输入：
- 原始的 `.ino` 代码或功能描述
- MQTT 需求（上传什么数据、下发什么指令）

输出：
- 完整的 MQTT 库文件
- 修改后的 `.ino` 主程序

---

## 核心能力

| 功能 | 说明 |
|------|------|
| WiFi 连接 | 自动初始化 WiFi 连接到指定 SSID |
| MQTT 连接 | 自动连接到 MQTT Broker（支持 emqx、阿里云等） |
| 数据上传 | 将传感器数据格式化并发布到 Topic |
| 指令下发 | 接收云端指令并映射到本地控制逻辑 |
| 断线重连 | 自动断线重连机制 |
| 多 Topic 支持 | 支持多个发布/订阅 Topic |

---

## 集成流程

### 工作流程

1. **用户提供需求**
   - 原始代码或功能描述
   - 上传数据格式
   - 下发指令定义

2. **AI 自动生成**
   - 标准化 MQTT 库
   - 修改主程序集成 MQTT 逻辑
   - 输出完整可用代码

3. **开发者部署**
   - 复制代码到 Arduino IDE
   - 修改 WiFi 和 MQTT 配置
   - 编译上传

---

## 技术特点

**设计原则：**
- 库与业务逻辑分离
- 异步事件驱动
- 自动断线重连
- 低内存占用（适配 ESP32）

**支持的平台：**
- ESP32 系列
- ESP8266
- 其他 Arduino 兼容板

---

## 与硬件项目设计 Skill 的配合

- **硬件设计阶段**：定义通信协议（Topic 格式、数据字段）
- **驱动开发阶段**：MQTT Skill 自动集成网络功能
- **PRD 文档**：通信协议已锁定，MQTT 集成一致性有保障
